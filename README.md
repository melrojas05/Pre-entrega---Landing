# Pre-entrega---Landing
Pagina dedicada al cuidado integral de mascotas (perros, gatos y aves)
El objetivo principal de la página web es presentar un centro de estética y bienestar animal que ofrece servicios especializados y productos saludables, guiando al usuario a traves de las distintas secciones para conocer los servicios que brindamos, leer reseñas de los clientes y facilitar la comunicacion con el negocio a través de un formulario de contacto
